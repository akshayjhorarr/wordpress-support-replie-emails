I completely understand the frustration — the new Gutenberg layout caught a lot of people off guard, and the mobile experience is genuinely difficult.
Let me give you a few options depending on what works best for you:
Option 1 — Switch back to the Classic Editor (Easiest fix)
Install the free "Classic Editor" plugin from the WordPress plugin directory.
Go to Settings → Writing → select Classic Editor as default.
This immediately restores the old scrollable editor you're used to, and it works perfectly on all devices. This plugin is officially supported by WordPress.
Option 2 — Fix the Gutenberg layout without a plugin
In the Gutenberg editor, click the three-dot menu (⋮) in the top right corner.
Select Preferences → Panels.
You can pin the panels you use most so they're always visible without pulling them up.
Also try turning off Fullscreen mode from the same menu — this gives you more working space.
Option 3 — For editing on your phone
The WordPress mobile app (free on iOS and Android) is purpose-built for mobile editing. It's a much better experience than trying to use the admin panel in a phone browser, which was never officially supported.
My recommendation: Classic Editor plugin is the quickest solution and will feel immediately familiar. If you'd like to stick with Gutenberg long-term, the mobile app is the way to go for phone editing.
Let me know which option works for you!
